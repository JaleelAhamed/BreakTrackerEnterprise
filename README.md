# Break Tracker Enterprise

> A Python-based desktop application for employee productivity monitoring, automatic idle detection, break tracking, and report generation.

---

## Overview

Break Tracker Enterprise is a personal portfolio project developed to simulate an enterprise-level employee productivity monitoring system.

The application automatically detects system inactivity, records employee breaks, collects break reasons, generates professional Excel reports, and is designed to support centralized reporting for managers and HR teams.

The objective of this project is to demonstrate software architecture, desktop application development, automation, reporting, and enterprise application design using Python.
