# Break Tracker Enterprise

# Installation

1. Install Python.
2. Install required packages.

```bash
pip install -r requirements.txt
```

Run

```bash
python main.py
```

---

# First Launch

Enter

- Employee Name
- Employee ID
- Department
- Designation

Click Login.

---

# Daily Usage

Click Login.

The application starts monitoring automatically.

---

# Idle Detection

If no keyboard or mouse activity occurs for more than 3 minutes,

the application records idle time.

---

# Break Popup

When work resumes,

select the appropriate break reason.

---

# Logout

Click Logout.

The application automatically generates the daily report.
